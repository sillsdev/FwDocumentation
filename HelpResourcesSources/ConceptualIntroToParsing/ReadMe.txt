# ConceptualIntroToParsing #

This folder contains all the source files for the Introduction to Parsing document
available in FieldWorks under Help/Resources.

This document is to be edited in XMLMind XML Editor using XLingPaper.  The output is
generated as an HTML file and then checked in to:
